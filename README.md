# About this project
This project is Unity 3D project for simulation of a Indivison Theory in 3D.

# How to use
Install Unity 3D and open this project.
Click "Play" button to start simulation.

# How to build
Click "File" -> "Build Settings" -> "Build" button.

# Unity 3D version
2023.1.0b20
